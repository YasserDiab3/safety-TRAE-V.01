#### 1. ✅ عدم مسح أو استبدال البيانات
**المشكلة السابقة:**
- `saveToSheet()` كانت تحذف جميع الصفوف ثم تعيد كتابتها → خطر فقدان البيانات

**الحل المطبق:**

- الصفوف الجديدة تُضاف أسفل الشيت
- **لا يتم حذف أي صفوف إلا عند استدعاء `deleteRowById()` صراحةً**

**الملفات المعدلة:**

#### 2. ✅ التسجيل أسفل كل صف

**الملفات المعدلة:**

#### 3. ✅ تسجيل البيانات نصي/رقمي (بدون JSON)
**المشكلة السابقة:**
- `toSheetCellValue_()` كانت تستخدم `JSON.stringify()` للـ arrays/objects
- النتيجة: خلايا تحتوي على `{"key":"value"}` أو `["item1","item2"]`

**الحل المطبق:**
- **Arrays** → نص متعدد الأسطر (سطر لكل عنصر):
  ```
  item1
  item2
  item3
  ```
- **Objects** → نص بصيغة `key: value` على أسطر منفصلة:
  ```
  key1: value1
  key2: value2
  ```
- **الأرقام** → تبقى أرقام
- **التواريخ** → تبقى Date objects (تُعرض بشكل صحيح في Sheets)
- **النصوص** → تبقى نصوص

**الملفات المعدلة:**
- `Backend/Utils.gs` → دالة `toSheetCellValue_()` (السطور ~632-664)
